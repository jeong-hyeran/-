함수 콜백함수의 인자 콜백함수의 반환형 기능
array.forEach 배열 내부 객체 하나 void 제어문 for와 거의 동일하다.
배열 내 요소들을 이용하여 반복수행을 처리한다
array.sort 배열 내부 객체 둘 number 두 객체를 비교하여 결과를 정수로 반환
반환값이 0보다 크면 자리바꿈, 아니면 그대로 두기
array.filter 배열 내부 객체 하나 boolean 콜백함수의 반환형에 따라 새로운 배열에 포함시킬지 여부를 결정한다
array.map 배열 내부 객체 하나 object 콜백함수가 반환하는 객체형식으로 새로운 배열을 구성한다
